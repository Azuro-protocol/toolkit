# Toolkit

This framework-agnostic package provides essential utilities for building applications on the **Azuro Protocol**.


## Installation

```
npm i --save @azuro-org/toolkit
```

#### Peer Dependencies

```
@azuro-org/dictionaries@^3.0.28
graphql-tag@^2.12.6
viem@^2.37.4
```
